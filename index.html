Key information you may use

Hydraulics vs Pneumatics

  • Pneumatic system uses air or gas to transfer force.
  • Hydraulic system uses a liquid to transfer force.
  • Liquids are hard to compress, so hydraulics can create a strong push.
  • Hydraulic systems often use oil in real machines.

Task 7 – 3D Oblique Drawing (PAT page 8)
Use graph/grid paper. Oblique = front face stays flat; depth lines go back at 45°. Depth = 1 block. Pencil + ruler.
Perfect oblique method:
  1. Draw the front view first (dark outline).
  2. From each corner, draw a line back at 45°.
  3. Depth must be 1 block (one square).
  4. Connect the back edges to close the shape.
  5. Keep outlines dark and accurate.
